## نظرة عامة

مجموعة بيانات تدريب متخصصة بـ**محادثات البيع والشراء** للنماذج اللغوية (LLM) باللهجة العراقية الشعبية.  
تحتوي على **100,000 محادثة** موزعة على 10 ملفات، كل ملف يحتوي على **10,000 محادثة**.

**التخصص الأساسي:** محادثات البيع والشراء بين البائع والزبون بالسوق العراقي —  
التساؤل عن السعر، التفاوض، الخصم، الضمان، التقسيط، والإغلاق.  
البيانات مبنية على اللهجة العراقية البغدادية الشعبية وتشمل مفردات وتعابير من مدن عراقية مختلفة.

## مواضيع المحادثات

### محادثات البيع — النواة الأساسية للبيانات (80%)

كل محادثة بيع تدور بين **زبون عراقي** و**بائع عراقي** وتغطي سيناريوهات حقيقية من السوق:

| السيناريو | مثال |
|-----------|-------|
| الاستفسار عن المنتج | "هلو، عندكم سامسونج S24؟" |
| السؤال عن السعر | "بيش سعره؟ / شكد؟" |
| التفاوض والخصم | "ما تنزل شوية؟ / تعطيني تخفيض؟" |
| رفض السعر | "هذا غالي هواي / مو معقول هالسعر" |
| مقارنة المنتجات | "شنو الفرق بين هذا وذاك؟" |
| السؤال عن الضمان | "عنده ضمان؟ كم سنة؟" |
| الشراء بالتقسيط | "أكدر أقسطه؟ شكد المقدم؟" |
| الشراء بالجملة | "أريد 10 قطع، تعطيني سعر خاص؟" |

## بنك المفردات — RAG Reference (word.json)

> **ملف RAG دائم** — يُستخدم كقاعدة معرفة للاسترجاع المعزز (Retrieval-Augmented Generation).  
> يُحمَّل أثناء التوليد ليضمن أن كل محادثة تستخدم مفردات عراقية أصيلة.  
> المصدر: مراجعة يدوية + [معجم اللهجة العراقية](https://ar.mo3jam.com/dialect/Iraqi)

### الاستخدام كـ RAG

```python
import json

with open("word.json", encoding="utf-8") as f:
    vocab = json.load(f)

# البحث عن مفردة في تصنيف معين
def get_category(name):
    return next((c["items"] for c in vocab if c["category"] == name), [])

# مثال: جلب كلمات التحية
greetings = get_category("التحية")
```

# بيانات تدريب اللهجة العراقية
## Iraqi Arabic Dialect — LLM Training Dataset

---

## نظرة عامة

مجموعة بيانات تدريب للنماذج اللغوية (LLM) باللهجة العراقية الشعبية.  
تحتوي على **10,000 محادثة** موزعة على 10 ملفات، كل ملف يحتوي على **1,000 محادثة**.

---

| الملف | الموضوع | المحادثات |
|-------|---------|-----------|
| `iraqi_train_01_electronics.json` | بيع الإلكترونيات (موبايل، لابتوب، تلفزيون) | 1,000 |
| `iraqi_train_02_food.json` | بيع المواد الغذائية والخضار | 1,000 |
| `iraqi_train_03_clothes.json` | بيع الملابس والأحذية | 1,000 |
| `iraqi_train_04_cars.json` | بيع السيارات | 1,000 |
| `iraqi_train_05_realestate.json` | بيع وإيجار العقارات | 1,000 |
| `iraqi_train_06_furniture.json` | بيع الأثاث والأجهزة المنزلية | 1,000 |
| `iraqi_train_07_services.json` | الخدمات (تصليح، تركيب، صيانة) | 1,000 |
| `iraqi_train_08_daily.json` | محادثات يومية عامة | 1,000 |
| `iraqi_train_09_social.json` | محادثات عائلية واجتماعية | 1,000 |
| `iraqi_train_10_mixed.json` | مزيج من كل الأنواع | 1,000 |

## صيغة البيانات

كل محادثة بصيغة JSON بهذا الشكل:

```json
{
  "id": "electronics_sales_0001",
  "category": "sales_electronics",
  "dialect": "iraqi_arabic",
  "messages": [
    { "role": "user",      "content": "السلام عليكم" },
    { "role": "assistant", "content": "وعليكم السلام، هلا وغلا! كيف أكدر أساعدك؟" },
    { "role": "user",      "content": "أريد موبايل سامسونج، عندكم؟" },
    { "role": "assistant", "content": "إي أكو، سامسونج S24 جديد وبالكرتون. سعره 750,000 دينار." }
  ]
}
```

## خصائص اللهجة العراقية

| الفصحى / الشامي | العراقي |
|-----------------|---------|
| كيف حالك | شلونك |
| ماذا / ما | شنو |
| بكم / كم | بيش / شكد |
| متى | شوكت |
| يمكن / تستطيع | تكدر / أكدر |
| نعم | إي / إي والله |
| موافق | ماشي |
| كثير | هواي |
| الآن | هسه |
| يوجد | أكو |
| لا يوجد | ما أكو |
| أعجبني | عجبني |
| أي (للسؤال) | يا |
| جاء / جاءت | اجه / اجت |
| معطل | عطلان |
| النقود | الفلوس |

## مواضيع المحادثات

**محادثات البيع:**
- التساؤل عن المنتج والسعر
- التفاوض على السعر والخصم
- مقارنة المنتجات
- الضمان والتقسيط
- الشراء والإغلاق

**المحادثات اليومية:**
- أحوال الطقس والكهرباء
- الصحة والمستشفيات
- العائلة والزواج
- الدراسة والعمل
- الطبخ والأكل
- الرياضة والترفيه
